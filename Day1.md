# Day 1 - Linux Learning Journey 🚀

## Commands Learned Today

### pwd

Show current working directory:

```bash
pwd
```

Example output:

```text
/home/user/LINUX-JOURNEY
```

---

### ls

List files and folders:

```bash
ls
```

Useful variations:

```bash
ls -l
ls -a
```

---

### cd

Change directory:

```bash
cd folder_name
```

Go back one directory:

```bash
cd ..
```

Go to home directory:

```bash
cd ~
```

---

### mkdir

Create a new directory:

```bash
mkdir folder_name
```

Create multiple folders:

```bash
mkdir Day-01 Day-02 Day-03
```

---

### touch

Create files:

```bash
touch file.txt
```

Create multiple files:

```bash
touch a.txt b.txt c.txt
```

---

### rm

Remove files:

```bash
rm file.txt
```

Remove folder recursively:

```bash
rm -r folder_name
```

Force remove:

```bash
rm -rf folder_name
```

---

### clear

Clear terminal screen:

```bash
clear
```

Shortcut:

```bash
Ctrl + L
```

---

## Practice Summary

- Learned directory navigation
- Created folders using `mkdir`
- Created files using `touch`
- Removed files using `rm`
- Explored directories using `pwd`, `ls`, and `cd`
- Cleared terminal using `clear`
